# LwIP

[![status][PIPELINE_STATUS]][PROJECT_URL]

This is a branch of liblwip with a simple build system.

## How to Build

**Unix**:
```bash
git clone https://gitlab.com/hev/lwip
cd lwip
make
```

**Android**:
```bash
mkdir lwip
cd lwip
git clone https://gitlab.com/hev/lwip jni
ndk-build
```
